Halo para pembaca, dalam post pertama ini, saya akan membahas tentang sebuah game engine untuk membuat suatu game. Game engine tersebut adalah Cocos2d-x.

Mungkin para pembaca masih ada yang asing dengan kata Game Engine . Game engine adalah sebuah software framework (kumpulan code) yang dipakai untuk mempermudah dalam proses pembuatan / pengembangan sebuah game. Karena tanpa adanya sebuah game engine, seorang developer (pengembang game) akan sangat kesulitan dalam proses pembuatan game.

Game engine yang biasa dikenal oleh para game developer antara lain: Unity, Construct2, GameMaker, XNA, Monogame, Cocos2d-x, dan lain sebagainya. Untuk post kali ini kita akan membahas tentang Cocos2d-x.

Cocos2d-x adalah sebuah game engine yang powerfull, mudah digunakan, dan game engine 2D open source. Open source disini maksudnya setiap orang dapat berkontribusi dalam pengembangan game engine ini. Dan tentu saja cocos2d-x ini 100% gratis. Cocos2d-x ini juga banyak dipakai oleh pengembang game ternama seperti Rovio, Bandai Namco, Square Enix, dan banyak lagi yang lainnya.
