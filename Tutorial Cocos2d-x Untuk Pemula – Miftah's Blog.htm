Tutorial Cocos2d-x Untuk Pemula

Halo para pembaca, dalam post pertama ini, saya akan membahas tentang sebuah game engine untuk membuat suatu game. Game engine tersebut adalah Cocos2d-x.

Mungkin para pembaca masih ada yang asing dengan kata “Game Engine”. Game engine adalah sebuah software framework (kumpulan code) yang dipakai untuk mempermudah dalam proses pembuatan / pengembangan sebuah game. Karena tanpa adanya sebuah game engine, seorang developer (pengembang game) akan sangat kesulitan dalam proses pembuatan game.

Game engine yang biasa dikenal oleh para game developer antara lain: Unity, Construct2, GameMaker, XNA, Monogame, Cocos2d-x, dan lain sebagainya. Untuk post kali ini kita akan membahas tentang Cocos2d-x.

Cocos2d-x adalah sebuah game engine yang powerfull, mudah digunakan, dan game engine 2D open source. Open source disini maksudnya setiap orang dapat berkontribusi dalam pengembangan game engine ini. Dan tentu saja cocos2d-x ini 100% gratis. Cocos2d-x ini juga banyak dipakai oleh pengembang game ternama seperti Rovio, Bandai Namco, Square Enix, dan banyak lagi yang lainnya.

Catatan : Saya sarankan sebaiknya anda sudah memahami dasar-dasar pemrograman yang baik, dan juga sudah terbiasa menggunakan bahasa pemrograman C++

Tahap Awal

Unduh versi terbaru dari Cocos2d-x di http://www.cocos2d-x.org/download

Taruh file yang telah diunduh tadi ke tempat yang kamu inginkan dan ekstrak. Saya sarankan, sebaiknya taruh file .zip yang telah diunduh ke tempat terluar dari sebuah drive.

Unduh juga Android NDK (Native Development Kit) di https://developer.android.com/ndk/downloads/index.html, dan jangan lupa pilih link unduhan yang sesuai dengan jenis komputermu. Jika komputermu berbasis 64 bit, pilih yang Windows 64-bit, jika 32 bit pilih yang Windows 32-bit.

Tujuan kita mengunduh android ndk ini adalah agar code-code kita dalam bahasa C++ bisa diproses untuk dicompile pada platform android.

Unduh juga Android SDK di http://android-sdk.en.softonic.com/

Unduh juga Apache Ant di http://ant.apache.org/bindownload.cgi

Unduh juga Phyton di https://www.python.org/downloads/

Jangan lupa, pilih phyton yang versi 2.x.x. Tujuan kita mengunduh phyton ini adalah karena semua perintah console cocos2d-x dijalankan menggunakan phyton.

Setelah semua file yang diperlukan sudah diunduh, sekarang ekstrak satu-persatu file-file yang sudah diunduh tadi. Saya sarankan taruh file-file yang diunduh tadi dalam satu folder untuk memudahkan kita ketika proses instalasi.

Kemudian install phyton yang telah diunduh tadi.

Setelah semua langkah tadi sudah dilakukan, langkah selanjutnya adalah proses menginstall cocos2d-x.

  1. Buka command prompt, lalu arahkan ke lokasi file cocos2d-x yang telah diunduh tadi. Misalkan, lokasinya ada di E:\cocos2d-x-3.12, maka ketikkan perintah di command prompt
  2. Lalu ketikkan perintah setup.py. Maka cocos2d-x akan secara otomatis mencatatkan dirinya ke dalam Environment Variable. Kemudian akan muncul tampilan seperti di bawah ini. Lalu, masukkan lokasi / path folder android ndk yang sudah diunduh tadi.
  3. Kemudian akan muncul tampilan di bawah ini, lalu masukkan juga lokasi / path folder android sdk.
  4. Kemudian masukkan juga lokasi / path folder Apache Ant.
  5. Jika semua langkah diatas telah dilakukan, berarti cocos2d-x sudah terinstall dengan baik. Untuk cara mengeceknya adalah ketikkan perintah cocos di command prompt, jika tampilannya seperti ini, maka proses instalasi telah berhasil.

Selamat! sekarang anda sudah bisa menggunakan game engine cocos2d-x ini. Meskipun cara installnya cukup ruwet dan susah, tapi cocos2d-x ini sangat powerfull untuk membuat game dua dimensi. Di postingan selanjutnya, akan saya bahas cara memulai sebuah project game cocos2d-x baru.
